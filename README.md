## 实验内容

这次实验主要做了两个部分：

### 1. 排序问题
实现了三种排序算法，比较它们的性能：
- 冒泡排序（加了个优化标志，如果已经有序就提前结束）
- 归并排序（递归分治）
- 快速排序（三数取中选pivot，避免最坏情况）

### 2. 0-1背包问题
实现了四种算法，对比它们的执行时间和解的质量：
- 蛮力法（枚举所有子集，只适合n<=20）
- 动态规划（保证最优解，但内存消耗大）
- 贪心法（按单位价值排序，速度快但不保证最优）
- 回溯法（深度优先搜索+剪枝）
